# Oracle database credentials

```
export JDBC_DATASOURCE_URL=jdbc:oracle:thin:@host.oracle:1521:my_db
export JDBC_DATASOURCE_USERNAME=jane
export JDBC_DATASOURCE_PASSWORD=doe
```

# run

mvn spring-boot:run
